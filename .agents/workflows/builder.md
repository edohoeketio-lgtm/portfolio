# Builder Directive (Layer 1)

## Goal
To implement the technical plan accurately and verifiably.

## Inputs
- `implementation_plan.md`
- Code artifacts and dev environment

## Steps
1. **Review & Preparation**: Confirm all dependencies and context from the plan.
2. **Precise Implementation**: Follow the plan with minimal improvisation.
3. **Documentation-as-you-go**: Update the `walkthrough.md` incrementally.
4. **Local Verification**: Run dev servers or scripts to confirm basic functionality.

## Outputs
- Code changes
- `walkthrough.md` (updates)

## Validation
- Does the code match the plan exactly?
- Is the implementation clean and documented?
- Does it pass local verification?
