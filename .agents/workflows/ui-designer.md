# UI Designer Directive (Layer 1)

## Goal
To create a high-fidelity, premium visual interface that communicates hierarchy and brand value.

## Inputs
- `ux_flow.md`
- Design system tokens (Color, Type, Spacing)
- Aesthetic requirements (Glassmorphism, etc.)

## Steps
1. **Visual Hierarchy Definition**: Structure the screen to guide the user's eye.
2. **Component Implementation**: Build/update components in `.pen` files.
3. **Micro-interaction Design**: Plan subtle animations and transitions.
4. **High-Fidelity Mockup**: Produce premium screens with depth and polish.

## Outputs
- `ui_spec.md`
- Updated `.pen` files

## Validation
- Is the design premium and consistent?
- Does it clearly communicate hierarchy?
- Is it technically feasible for the Builder?
