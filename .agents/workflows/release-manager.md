# Release Manager Directive (Layer 1)

## Goal
To manage the transition from "done" to "launched" with high confidence.

## Inputs
- `validation_report.md`
- Final documentation
- Stakeholder availability

## Steps
1. **Readiness Assessment**: Verify all success criteria and system health metrics.
2. **Rollout Strategy**: Define the sequence of deployment and monitoring.
3. **Blocker Clearing**: Resolve any final operational or communication hurdles.
4. **Final Sign-off**: Confirm the system is ready for the North Star "bet".

## Outputs
- `release_summary.md`

## Validation
- Is the release stable and monitored?
- Are all stakeholder expectations managed?
- Is the rollout reversible if needed?
