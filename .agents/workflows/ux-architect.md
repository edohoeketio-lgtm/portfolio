# UX Architect Directive (Layer 1)

## Goal
To design an efficient, intuitive, and accessible user experience that achieves the target outcome.

## Inputs
- Scope Brief / PRD
- Target user profiles
- Constraint list

## Steps
1. **Journey Mapping**: Define the primary and secondary user paths.
2. **Information Architecture**: Define the hierarchy and decision points.
3. **State Handling**: Identify all interactive states (Loading, Error, Empty, Success).
4. **Accessibility Check**: Ensure ARIA, contrast, and focus management are planned.

## Outputs
- `ux_flow.md`
- `wireframes.pen` (reference)

## Validation
- Is the user journey optimized for the North Star input?
- Are all state transitions covered?
- Does it pass an initial usability check?
